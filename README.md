# Portafolio - Modelos de Evaluación de Recursos Educativos Digitales

Este repositorio contiene un sitio web académico que replica el diseño del portafolio de evaluación de Recursos Educativos Digitales (RED), desarrollado como parte de la **Especialización en Aplicación de TIC para la Enseñanza – EATICE** de la Universidad de Santander.

## Contenido

El sitio consta de las siguientes páginas:

- **Inicio**: Presentación del autor, título, profesor y universidad, con una introducción al trabajo.
- **Modelos de evaluación y RED**: Tabla comparativa de seis modelos de evaluación (LORI, Reeves, CODA, Norma UNE 71362, FURPS, McCall) y explicación de cada uno. Incluye dos subpáginas:
  - *Seleccionando modelos*: Justificación de la elección de CODA, LORI y Reeves, y video de sustentación.
  - *Recursos Educativos Digitales*: Fichas detalladas de los dos RED evaluados (Rural STEAM Lab y PhET Interactive Simulations).
- **Rediseño**: Propuesta del modelo LORI-RSL con nuevos indicadores, diagrama y justificación.
- **Aplicación del modelo**: Resultados de la aplicación de CODA, LORI y Reeves, comparación entre modelos, selección del modelo más adecuado y aplicación de LORI-RSL.
- **Referencias**: Lista de referencias bibliográficas en formato APA.

## Tecnologías utilizadas

- HTML5
- CSS3 (sin frameworks, diseño responsive)
- JavaScript puro
